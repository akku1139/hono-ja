# その他

## 貢献

貢献を歓迎しています！以下の方法で貢献できます。

- Issue を作成する - 新しい機能を提案する。 バグを報告する。
- プルリクエスト - バグやタイプミスを修正する。 コードをリファクタリングする。
- サードパーティ製ミドルウェアを作成する - 下記にリンクがあります。
- 共有する - ブログ, X(Twitter), 他の媒体であなたの考えを共有します。
- アプリケーションを作成する - Hono を使用してみてください

詳細は [Contribution Guide](https://github.com/honojs/hono/blob/main/docs/CONTRIBUTING.md) を参照してください。
